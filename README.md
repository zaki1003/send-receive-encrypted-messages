# send/receive encrypted messages
 send/receive encrypted messages using java sockets, RC4 Algorithm, RSA

## Getting Started
1. Install Java.
2. Run ServerSide.
3. Enter RC4 KEY and message.
4. Run ClientSide.

### Prerequisites
1. Knowledge of RC4 and RSA algorithms
2. Working knowledge of sockets
